# Express + TypeScript CRUD Server

- [x] Implement CRUD for user
- [x] Middleware for API request (validation)
- [x] Middleware for API response
- [x] Swagger

## Requirements

- Node.js >= 20 recommended
- npm or yarn


## ⚙️ Setup Instructions

### 1️⃣ Install Dependencies
```bash
npm install
```

### 2️⃣ Run Migration script

```bash
npm run migrate
```

### 2️⃣ Run Migration script


```bash
npm run dev
```

Test api at http://localhost:4000/api-docs/
